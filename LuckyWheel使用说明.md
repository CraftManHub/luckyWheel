# LuckyWheel · 使用说明

## 快速开始

1. 运行 `LuckyWheel.exe`
2. 在右侧面板确认/编辑抽奖选项
3. 点击底部「**开启仪式**」按钮，转盘开始旋转
4. 旋转停止后，结果显示在按钮下方

---

## 界面布局

```
┌─────────────────────────────┬──────────────┐
│                             │  xxxxxx      │
│        魔法转盘              │  选项列表      │
│        （点击画面            │  ＋放一个      │
│         触发花瓣特效）        │  －挪走它！    │
│                             │  ──────────  │
│                        ⚙  ← │  开启仪式     │
│                             │  结果显示     │
└─────────────────────────────┴──────────────┘
```

- **左侧**：转盘主体
- **右侧**：选项配置面板
- **右边缘的 ⚙ 按钮**：点击滑出背景/音效设置抽屉

---

## 管理抽奖选项

### 添加 / 删除

| 操作 | 说明 |
|------|------|
| **＋ 放一个** | 新增一行，直接输入选项名称 |
| **－ 挪走它！** | 删除当前选中行（至少保留 2 个） |

### 修改文字颜色

点击选项所在行的**颜色色块**（第二列），弹出颜色选择器，选好后确认即可。

---

## 配比（权重）

点击面板顶部的「**配比**」按钮展开权重列。

- 数字越大，被抽中的概率越高
- 转盘扇形大小与权重成正比
- 再次点击（变为「隐藏」）可收起该列

**示例：**

| 选项 | 配比 | 实际概率 |
|------|------|---------|
| 一等奖 | 10 | 10 / (10+30+60) = **10%** |
| 二等奖 | 30 | **30%** |
| 谢谢参与 | 60 | **60%** |

---

## 份数（库存上限）

点击面板顶部的「**份数**」按钮展开份数列。

| 份数值 | 含义 |
|--------|------|
| `∞` | （输入-1可以设置）无限次，永远不会耗尽（默认） |
| 正整数（如 `3`） | 最多可中 3 次，每次中奖自动扣 1 |
| `0` | 已耗尽，下次旋转自动从转盘移除 |

**「不管份数」按钮**：激活后临时忽略所有份数限制，适合彩排演练。

---

## 背景 & 音效设置

点击右边缘的 **⚙** 按钮滑出设置抽屉。

### 背景图

| 设置项 | 作用 |
|--------|------|
| **转盘卷轴** | 转盘圆形区域内的背景图 |
| **世界卷轴** | 整个窗口的背景图 |
| **✕ 撕毁** | 恢复为默认渐变色 |

- 支持格式：JPG / PNG / GIF（动态）/ BMP
- 背景路径自动保存，重启后自动恢复

### 音效

将音频文件放入程序同级的 `sounds/` 文件夹，转盘每次停止后随机播放一个。

- 支持格式：`mp3` `wav` `ogg` `flac` `m4a` `aac`
- 点击「**嘘，别出声**」按钮切换静音

---

## 其他交互

- **拖动窗口**：在任意空白处按住鼠标拖动即可移动窗口
- **花瓣特效**：点击画面任意位置触发花瓣/心形/星星粒子爆发

---

## 配置自动保存

| 内容 | 保存位置 |
|------|---------|
| 抽奖选项、权重、份数、颜色 | `%APPDATA%\LuckyWheel\config.json` |
| 背景图路径 | `%APPDATA%\LuckyWheel\bg_config.json` |

> 移动背景图的路径后需重新选择背景图。
